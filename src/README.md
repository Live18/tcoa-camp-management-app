
# Basketball Camp Admin System

This project is a comprehensive management system for basketball camps, with features for managing users, admins, locations, games, and classroom sessions.

## Project Structure

- **Admin Management**: A system for managing administrators with Super Admin and Regular Admin roles
- **User Management**: Tools for managing campers, observers, and presenters
- **Location Management**: Facilities for managing camp locations
- **Game Management**: System for scheduling and managing games
- **Session Management**: Tools for organizing classroom sessions

## Key Features

- **Role-Based Access Control**: Granular permissions system based on user roles
- **Super Admin Capabilities**: Special privileges for system administrators
- **Admin Logging**: Complete audit trail of administrative actions
- **User Assignment**: Tools for assigning users to games and sessions
- **Notification System**: Email and SMS notifications for important events

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Shadcn UI Components

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
